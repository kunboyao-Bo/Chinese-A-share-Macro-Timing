# 大盘宏观择时策略 | Macro Timing for Chinese A-Share Market

一个基于宏观经济指标的**中国A股大盘择时系统**，包含信号测试、实时信号生成和完整回测框架。

------------------------------------------------------------------------------

## 📖 项目介绍

本项目通过**宏观经济 + 估值 + 市场情绪 + 动量过滤**的组合，构建了一个结构化的大盘择时模型，用于判断市场未来1-3个月的趋势，并输出清晰的仓位建议。

**核心目标**：在控制回撤的同时，捕捉市场的主要上涨阶段。

------------------------------------------------------------------------------

## 包含的脚本

### 1. `宏观择时IC检验.py`
- 对多种宏观信号进行**信息系数（IC）测试**
- 支持 Pearson 和 Spearman 相关性
- 输出 IC 均值、ICIR、t统计量、p值等
- 测试信号包括：PMI动量、PPI、PE分位、成交额Z-score等
- 自变量包括ret2m、ret3m、ret12m

### 2.宏观择时模型包括2种（有无momentum buffer）
### 2.1 `宏观择时模型.py`
- **核心交易信号生成模型**
- 使用三个核心因子：PE历史分位 + PPI + 成交额Z-score
- 输出信号：**强烈买入 / 买入 / 不操作 / 卖出 / 强烈卖出**

### 2.2 `宏观择时模型_带动量筛选.py`
- **核心交易信号生成模型**
- 使用三个核心因子：PE历史分位 + PPI + 成交额Z-score
- 加入**动量过滤器**（OR逻辑），避免在强趋势中逆势操作
- 输出信号：**强烈买入 / 买入 / 不操作 / 卖出 / 强烈卖出**

### 3. `宏观择时回测.py`
- 完整历史回测引擎
- 次月开盘价执行，考虑交易成本（买入0.02%，卖出0.07%）
- 仓位映射：100% / 70% / 30% / 10% / 0%
- 输出净值曲线、仓位变化、回撤、绩效指标（年化收益、Sharpe、Calmar、最大回撤等）
- 包含信号预测显著性检验（t检验）

---

## 🛠️ 主要信号逻辑

| 信号 | 含义 | 得分逻辑 |
|------|------|---------|
| PE历史分位 | 估值水平 | <30% → +1（低估），>70% → -1（高估） |
| PPI | 通胀/经济周期 | <-1% → +1，>3% → -1 |
| 成交额Z-score | 市场情绪 | <-0.8 → +1（冰点），>1.2 → -1（过热） |
| 动量过滤器 | 趋势确认 | 12个月上涨月数≥7 或 12个月收益>-20% |
